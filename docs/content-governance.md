# Content Governance

This recap is a public-facing editorial artifact. Treat updates as reviewed publication changes, not scratch edits.

## Source Discipline

- Speaker names, company names, titles, prize amounts, and session claims should be checked against source transcripts, event notes, or approved stakeholder corrections.
- If a transcript label is mistranscribed, keep the corrected public name in the recap and document the correction in the commit message or source notes.
- Do not add direct quotes unless the wording is verified from a transcript or approved source.
- Paraphrases should stay clearly grounded in the session content.

## Asset Discipline

- Keep published images in `images/`.
- Use descriptive file names for new assets.
- Before publishing, verify that the person, session, and company shown in a photo match the section where it appears.
- Do not replace winner, speaker, or keynote photos without checking the page in browser preview.

## Review Checklist

- All tabs open and show exactly one active panel.
- No broken images.
- No old placeholder names such as `Mhire`.
- No removed sections such as `Video Script`.
- Mobile layout has no horizontal page overflow.
- Public URL loads from a clean browser session.
