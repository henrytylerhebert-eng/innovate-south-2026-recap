# Innovate South 2026 Newsletter Package

Mailchimp-ready materials for promoting the Innovate South 2026 recap.

## Files

- `launch-email.md` - reviewable newsletter copy, subject lines, preview text, and CTA map.
- `segment-versions.md` - audience-specific framing for founders, investors, partners, and attendees.
- `utm-links.csv` - tracked links back to the live recap.
- `startup-scoop-format.md` - reusable OM newsletter format based on the current Mailchimp/Canva pattern.
- `issue-intake-template.md` - planning packet for situational issues, current events, programming, ecosystem opportunities, jobs, and partner updates.
- `review-checklist.md` - pre-send QA checklist for source state, Canva assets, Mailchimp build, links, and approvals.
- `mailchimp/build-guide.md` - step-by-step Mailchimp assembly instructions.
- `mailchimp/email-safe-template.html` - optional code-your-own HTML version.

## Recommended Use

Use `launch-email.md` as the source of truth for copy, then assemble the email in Mailchimp's normal email builder with content blocks. Use the HTML template only if the team wants a code-your-own campaign and is comfortable testing it carefully.

For recurring Startup Scoop issues, start with `issue-intake-template.md`, draft against `startup-scoop-format.md`, then use `review-checklist.md` before scheduling in Mailchimp.

Mailchimp requires an unsubscribe link in all emails, and code-your-own templates should include the `*|UNSUB|*` merge tag. Keep Mailchimp's default footer unless there is a specific reason to customize it.
