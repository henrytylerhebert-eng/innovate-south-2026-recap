## Recommended Improvements While Rebuilding

- Keep deadlines and eligibility in live text, not only in graphics.
- Add alt text to every Canva graphic.
- Use one CTA button per module.
- Convert the jobs section into a tighter company-grouped list.
- Keep resource and mentor modules as two-column spotlight blocks on desktop, but preview mobile carefully.
- Use UTM links for OM-owned destinations if this issue will be resent or used as a template.

## Mobile QA

- Header logo and contact block should stack cleanly.
- Top image nav should not become unreadably small.
- Two-column resource/FlyGuys blocks should stack image first or text first intentionally.
- Buttons should be full-width or comfortably tappable.
- Job list should remain scannable.

## Compliance QA

- Include unsubscribe link.
- Include preference/update profile link.
- Include mailing address.
- Include archive/browser-view link.
- Do not remove Mailchimp compliance footer unless replacing it with valid merge tags.
